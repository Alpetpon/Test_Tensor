Для обхода и доступа к файловой системе будем Использование SSH 

Начнем с того, что установим SSH-клиент, после установки клиента нужно настроить доступ по ssh 

На большинстве дистрибутивов Linux SSH-клиент установлен по умолчанию. Но Для проверки можно использовать команду ssh -V

Если SSH-клиент отсутствует, его можно установить с помощью пакетного менеджера(Рассматриваем на примере Ubuntu)

Выполняем команды для установки 

sudo apt update
sudo apt install openssh-client

Для подключения к серверу по SSH нужно знать:
 - IP-адрес или DNS-имя сервера
 - Имя пользователя
 - Пароль или использовать ключи SSH для аутентификации
  
При использовании аутентификация по ключу, необходимо сгенерировать SSH-ключ и добавить публичный ключ в файл ~/.ssh/authorized_keys на
сервере.

После настройки доступа можно подключаться  к серверу, с помощью команды  ssh user@server_address

где user — имя пользователя на сервере, а server_address — IP-адрес или DNS-имя сервера.

После успешного подключения можно выполнять нужные операции с файловой системой

После выполнения необходимых операций необходимо завершить сеанс SSH, использовав команду exit
